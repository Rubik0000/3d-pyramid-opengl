## Tutorial
### Rotation
Use &uarr; &darr; to rotate figure around X axe<br/>
Use &larr; &rarr; to rotate figure around Z axe<br/>
Use keys **z** and **x** to rotate figure around Y axe<br/>
### Commands
Type those command on the keyboard directly on the screen
- **h** + \<number> + **ENTER**: define height of the pyramid<br/>
- **r** + \<number> + **ENTER**: define radius of the base<br/>
- **c** + \<number> + **ENTER**: define count of faces<br/>
- **o** + \<number> + **ENTER**: define opacity of the figure, the real value will be corresponding to \<number> / 10, e. g. to define 0.5 opacity you have to type 5 in command<br/>
- **ESC** to clear command buffer
